# sistema-clinica-java

## Descrição

Sistema de gerenciamento de pacientes desenvolvido em Java como atividade acadêmica da disciplina de Programação Orientada a Objetos. O projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais da orientação a objetos, desenvolvimento de interfaces gráficas e persistência de dados utilizando banco de dados relacional.

A aplicação permite realizar operações de cadastro, consulta, edição e remoção de pacientes, bem como o gerenciamento dos procedimentos associados a cada registro.

---

## Objetivos

- Aplicar os princípios da Programação Orientada a Objetos.
- Desenvolver uma aplicação desktop utilizando Java Swing.
- Implementar operações de persistência de dados utilizando JDBC e SQLite.
- Modelar entidades e relacionamentos entre objetos.
- Consolidar conhecimentos sobre desenvolvimento de aplicações Java.

---

## Funcionalidades

- Cadastro de pacientes
- Consulta de pacientes por CPF
- Atualização de informações cadastrais
- Remoção de pacientes
- Associação de procedimentos aos pacientes
- Persistência dos dados em banco SQLite

---

## Tecnologias Utilizadas

- Java
- Java Swing
- JDBC
